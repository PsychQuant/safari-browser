## 3. SaveImageCommand + registry

- [x] 3.1 實作「CLI surface: `--element` required, multi-match mirrors #30」滿足「--element is required and multi-match is fail-closed with --element-index disambiguation」需求 — 新 `Sources/SafariBrowser/Commands/SaveImageCommand.swift`：`@Argument var path` + `@Option var element: String` (required via validate())+ `@Option var elementIndex: Int?` + `@Option var track: String = "currentSrc"` + `@Flag var withCookies = false` + `@OptionGroup var target: TargetOptions`；`validate()` 檢查 elementIndex 要求 element、`elementIndex >= 1`、`track ∈ {src, currentSrc, poster}`；multi-match 行為直接 reuse `SafariBridge.resolveElementResource` 的 fail-closed + ambiguous + index-out-of-range 錯誤路徑
- [x] 3.2 Download dispatcher — `run()` 流程：`resolveNativeTarget` → `SafariBridge.resolveElementResource` → switch on `ElementResource` kind：`.inlineSVG(html)` 直接 UTF-8 寫檔；`.url(URL)` 依 scheme 分流（http(s) → URLSession / data: → Foundation decode / `--with-cookies` → fetchResourceWithCookies / 其他 → throw unsupportedURLScheme）；一條錯誤路徑都不寫檔
- [x] 3.3 實作「URL request: Referer + Safari UA, drop on cross-origin redirect」滿足「URL requests include Referer and Safari User-Agent, drop Referer on cross-origin redirect」需求 — URLSession request header 設 `Referer = doJavaScript("document.URL")` + `User-Agent = doJavaScript("navigator.userAgent")`；實作 `URLSessionTaskDelegate.urlSession(_:task:willPerformHTTPRedirection:newRequest:completionHandler:)` 檢測 cross-origin（scheme/host/port 任一不同）drop Referer，同 origin 保留
- [x] 3.4 實作「URL scheme policy: http(s) + data: supported, others fail-closed」滿足「URL scheme policy supports http(s) and data:, fails closed on others」需求 — scheme switch：`http` → stderr warn + URLSession；`https` → URLSession；`data:` → split-on-first-comma + base64 decode + 寫檔；其他（blob/ftp/javascript/file/未知）→ throw `unsupportedURLScheme`。同時實作「Output path: pure bytes-to-path, zero MIME inference」滿足「Output path is trusted verbatim with no MIME inference」需求 — 所有路徑共用「bytes → user path」單一寫檔函式，不讀 Content-Type、不查 MIME 表、overwrite 靜默
